# DNS-Tip-HTML
有好的DNS拦截提示页面

## 如何使用
clone 本项目到本地 修改 index.html 中的ip信息，域名信息，提示信息。

## 特点
 - 使用 js 判断，不同的域名、ip 显示不同信息
 - 全端适配，页面在手机、PC pad 均可美观展示
 - 适配夜间模式（跟随系统，需要系统支持）
